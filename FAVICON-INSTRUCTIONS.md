# Favicon Setup Instructions

## Status
✅ **Favicon links added to all 33 HTML pages**

## What You Need to Do

1. **Save your diamond logo image** as `favicon.png` in this directory:
   ```
   d:\Project-1\Gemstone-webisite\website-ver-1\aurex-gem-house\favicon.png
   ```

2. **Image Requirements:**
   - **Format:** PNG
   - **Recommended size:** 512x512px (can be resized if needed)
   - **File name:** Must be exactly `favicon.png`

## What's Already Done

All HTML files now include these lines in the `<head>` section:
```html
<link rel="icon" type="image/png" sizes="512x512" href="favicon.png" />
<link rel="shortcut icon" href="favicon.png" />
```

## Testing

Once you add the `favicon.png` file:
1. Open any page in your browser
2. Look for the favicon in the browser tab
3. Clear browser cache if it doesn't appear immediately (Ctrl+F5)
